Букмекерська контора — демо

            
Реальна механіка ставок, прорахована до копійки

            
ReactJavaScriptState logicUI/UX

          

          

        

        

          

            
Frontend · клієнтська логіка

            
            
Огляд

            
Демонстраційна платформа букмекерської контори, що відтворює ключову механіку реального сервісу: перегляд подій, формування купона ставки, розрахунок потенційного виграшу за коефіцієнтами та керування балансом користувача. Мета проєкту — показати роботу зі складною бізнес-логікою на боці клієнта.

            
Що реалізовано

            
    
              
  • Каталог подій з коефіцієнтами та категоріями
    • 
              
  • Купон ставки — додавання кількох подій, перерахунок суми та виграшу в реальному часі
    • 
              
  • Логіка балансу: списання при ставці, нарахування при виграші
    • 
              
  • Стан застосунку керується через React-state без зовнішніх залежностей
    • 
              
  • Адаптивний інтерфейс під десктоп і мобільні
    • 
            

            
Роль

            
Повна реалізація фронтенду та клієнтської логіки — від структури компонентів до математики коефіцієнтів.

03

          

            
Дипломна робота: мережевий аналізатор з AI

            
Сама знаходить проблему в мережі — і сама її лагодить

            
PythonGemini APIMQTTSQLiteTelegram Bot

          

          

        

        

          

            
Дипломна · AI + Hardware

            
            
Огляд

            
Інтелектуальна система моніторингу, діагностики та авто-відновлення мережі. Програма постійно стежить за станом мережі, виявляє несправності й застосовує відповідні виправлення — самостійно, без участі користувача. У ядрі — гібридний AI-двигун з трирівневим резервуванням.

            
Архітектура (три компоненти)

            
    
              
  • Desktop Agent — Python 3.11 з графічним інтерфейсом (CustomTkinter), сканування мережі та логіка виправлень
    • 
              
  • Hardware Probe — апаратний зонд, що передає дані по MQTT для незалежної перевірки стану каналу
    • 
              
  • Telegram-бот — віддалене керування та сповіщення зі списком дозволених користувачів (whitelist)
    • 
            

            
Ключові можливості

            
    
              
  • Гібридний AI-двигун з трирівневим fallback: Gemini API → локальна база на 200+ сценаріїв → апаратний зонд
    • 
              
  • 35+ сценаріїв авто-виправлень мережевих проблем
    • 
              
  • Сканування мережі (ARP), діагностика DNS, керування пріоритезацією трафіку (QoS)
    • 
              
  • Локальне збереження стану та історії в SQLite
    • 
              
  • Поставка як готовий Windows .exe (PyInstaller)
    • 
            

            
Роль

            
Повна розробка всіх трьох компонентів: десктоп-агента, логіки AI-двигуна, інтеграції апаратного зонда та Telegram-бота. Підготовлено повний пакет матеріалів захисту.

          

        

      


      
      

        

          
04

          

            
TikTok → YouTube автопостинг

            
Завантажує й публікує відео замість тебе — за розкладом

            
Python · FastAPITelegram Mini AppYouTube Data APIOAuth 2.0VPS

          

          

        

        

          

            
SaaS · автоматизація

            
            
Огляд

            
SaaS-сервіс у форматі Telegram Mini App для автоматичного постингу відео на YouTube, TikTok та інші платформи. Підходить для перегону трафіку й повної автоматизації публікацій — користувач підключає акаунти, ставить відео в чергу, а сервіс публікує їх за розкладом.

            
Що реалізовано

            
    
              
  • Бекенд на FastAPI, розгорнутий на власному VPS
    • 
              
  • OAuth 2.0 авторизація акаунтів і автоматичне оновлення токенів
    • 
              
  • Планувальник публікацій з окремими слотами розкладу під кожен акаунт
    • 
              
  • Збір реальної статистики переглядів через YouTube Data API v3
    • 
              
  • Завантаження великих відео частинами (chunked upload) зі відстеженням статусу
    • 
              
  • Дашборд з чергою, метриками та топ-відео
    • 
            

            
Роль

            
Повна розробка бекенду, інтеграцій з платформами, логіки планувальника та інтерфейсу Mini App. Контроль усього циклу — від схеми бази до публікації відео.

          

        

      


    

  







  

    
Telegram-боти

    
Боти, що працюють за мене

    
Окремий напрям — Telegram-боти під реальні бізнес-задачі: платежі, контент, трейдинг.

    

      

        

          

          

            
Бот-перехідник з оплатою

            
Приймає оплату й після успішного платежу автоматично перекидає користувача в Telegram-застосунок. Повний платіжний флоу без участі адміністратора.

          

        

        

          

          

            
Бот криптокурсу

            
Видає навчальний курс із криптовалюти прямо в чаті: уроки крок за кроком, контроль проходження, доступ за статусом користувача.

          

        

        

          

          

            
Бот-трейдер з прогнозами

            
Підключений до біржі через API: аналізує сотні пар у реальному часі й надсилає трейдерам сигнали з точками входу, стоп-лосом, тейк-профітом та рівнем впевненості.
